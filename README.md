## 📚 6 Parts — 30 Questions

### Part 1 — Basic Operations & Arithmetic
Fundamental Python operations — variables, arithmetic, calculations.

| Q | Topic |
|---|-------|
| Q1 | Score Board Operator — addition |
| Q2 | Nutrition Bar Ingredient Chart — percentages |
| Q3 | Model Accuracy Calculation — division |
| Q4 | Arithmetic Operations — add, subtract, multiply, divide |
| Q5 | Temperature Conversion — Celsius to Fahrenheit |
| Q6 | Simple Interest Calculation |

### Part 2 — Conditional Statements
`if-else` statements and comparison operators.

### Part 3 — Loops & Iterations
`for` loops and `while` loops for repetitive tasks.

### Part 4 — Data Structures
Lists, Tuples, Sets, and Dictionaries.

### Part 5 — Functions
Function definition, parameters, return statements, and scope.

### Part 6 — Exception Handling & OOP
`try-except` blocks, Classes, Objects, and Inheritance.
